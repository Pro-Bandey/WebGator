# WebGator

## Changelog


## v1.0 — Initial Prototype
- Floating navigation bar injected via content script
- Buttons: **Back, Forward, Reload**
- Appears when cursor reaches **top-center**
- Desktop-only detection (screen width + pointer)
- Basic fade and slide animation

- Glassmorphism background (blur + transparency)
